# Log Analyzer

## Мақсаты
Файлдан лог мәліметтерін оқып, ERROR, INFO, WARNING санын анықтау.

## Қолданылған технология
- Java
- BufferedReader
- FileReader
- Exception Handling

## Сұрақтарға жауап

### 1. Файл қалай оқылады?
Файл BufferedReader арқылы жол-жолымен оқылады.

### 2. Exception не үшін қажет?
Қате шыққанда программаны тоқтатпай өңдеу үшін қажет.

### 3. Қандай мәліметтер талданады?
ERROR, INFO, WARNING хабарламалары талданады.
